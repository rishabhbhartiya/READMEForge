## ✦ What is ReadmeForge?

ReadmeForge generates **edge-rendered SVG components** for GitHub READMEs via a simple URL API. Every component is a plain `![alt](url)` markdown image — no JavaScript, no iframes, no GitHub Actions needed.

```md
<!-- Drop any ReadmeForge URL into your README like this: -->
![My Banner](https://readmeforge.natrajx.in/api/banner?text=Hello+World&metal=gold)
```

> **Built by [Natraj-X](https://www.natrajx.in/)** — an AI & IT engineering agency specialising in ML pipelines, data engineering, and full-stack product development.

## ✦ API Reference

### Base URL
```
https://readmeforge.natrajx.in/api/{component}?param=value
```
All routes return `image/svg+xml` · No auth required · Globally edge-cached

---

### 1. Banner — `/api/banner`

| Param | Type | Default | Description |
|---|---|---|---|
| `text` | string | `READMEFORGE` | Main heading text |
| `subtext` | string | — | Subtitle / tagline |
| `metal` | string | `chrome` | Metal theme name |
| `colors` | string | — | Comma-separated hex override e.g. `#ff0000,#0000ff` |
| `angle` | number | `135` | Gradient angle (0–360) |
| `type` | string | `wave` | Shape: `wave` `flat` `diagonal` `arch` `chevron` `egg` `venom` `wave-bottom` `shark` `slice` `cylinder` |
| `animation` | string | `none` | `shimmer` `pulse` `fadeIn` `scanline` `glow` `scan` `twinkling` |
| `width` | number | `800` | SVG width px |
| `height` | number | `160` | SVG height px |
| `theme` | string | `dark` | `dark` or `light` |
| `align` | string | `center` | `left` `center` `right` |
| `section` | string | — | `header` or `footer` (flips shape) |
| `fontFamily` | string | `Orbitron` | `Orbitron` `Rajdhani` `Share Tech Mono` `Arial Black` `Georgia` |
| `bg` | string | — | Background color override (hex) |

```md
![Banner](https://readmeforge.natrajx.in/api/banner?text=MY+PROJECT&subtext=Built+with+love&metal=neon-green&type=wave&width=900)
```

### 2. Stat Card — `/api/card` · `/api/card-neo` · `/api/card-glass`

| Param | Type | Default | Description |
|---|---|---|---|
| `title` | string | `GitHub Stats` | Card title |
| `value` | string | — | Main value |
| `metal` | string | `chrome` | Metal theme |
| `style` | string | `default` | Design style preset |
| `width` | number | `320` | Width px |
| `height` | number | `160` | Height px |
| `icon` | string | — | Icon identifier |
| `theme` | string | `dark` | `dark` or `light` |

`/api/card-neo` also accepts `neoTheme: dark | light`, `neoStyle: raised | inset | flat`
`/api/card-glass` also accepts `glassTheme: dark | light`
